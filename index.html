A little bit about me

Currently I'm working at Bad Dinosaur, a software agency in Edinburgh. Here I am currently leading a small design team while also doing day-to-day design work on client projects.

In the past, I've worked on product teams as well as freelanced. I'm used to working hard to get products out the door and into users' hands for testing.

I'm involved in the whole process of product design, from forming ideas, drawing out rough user flows, building prototypes, user testing - right up to using HTML/CSS/JavaScript to create views ready for back-end development.

I love teaching what I know. I mentor junior designers on visual design as well as front-end code and user experience. In my current role I've been sharing my knowledge of front-end code and design with the wider team, including back-end developers.
